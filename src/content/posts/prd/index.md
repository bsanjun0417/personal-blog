# PRD: 나나스빵집 단일 홈 웹사이트

## 목표
취미로 빵을 굽고 지인/모임에 나누는 “나나스빵집” 소개용 정적 웹사이트 제작.
![블로그 썸네일](/uploads/prd/nanas-bread-04.png)
## 기술
- HTML
- CSS
- JavaScript 약간
- React/Vue 사용하지 않음
- GitHub Pages 배포 가능 구조

## 파일 구조
/index.html
/style.css
/script.js
/assets/hero-bread.png

## 디자인
- 크림색 배경
- 딸기우유 핑크 포인트
- 브라운 텍스트
- 둥근 카드, 손그림 스티커 감성
- 모바일 우선 반응형

## 섹션
1. Header
- 로고: 나나스빵집
- 메뉴: 오늘의 빵 / 기록 / 문의

2. Hero
- 문구: 오늘도 나나가 구웠어요
- 설명: 취미로 굽고, 가끔 나누는 작은 홈베이커리
- 버튼: 오늘의 빵 보기

3. 오늘의 빵
- 소금빵, 휘낭시에, 식빵 카드

4. 빵 기록
- 사진 카드 느낌으로 최근 베이킹 기록 표시

5. 문의
- 인스타/카카오 문의 버튼
- 링크는 임시로 #

## JS 기능
- “오늘의 빵 랜덤 추천” 버튼
- 클릭 시 빵 이름과 짧은 문구 변경
